Changelog

更新公告

内核重构 · 5 任务独立并行 · 子 Agent 系统 · 代码理解增强

v1.2.0 2026-07-10

本次更新亮点

  • 7 大国产模型厂商 — DeepSeek、通义千问、智谱清言、豆包、Kimi、StepFun、MiniMax，一键切换，独立 API Key
  • 10+ 代码理解工具 — get_file_outline、find_symbol、find_references、trace_symbol、get_file_imports、find_related_files、search_symbols、build_code_index、scan_project 等
  • 子 Agent 系统 — spawn_subagent 支持 6 类专家：explore（只读调研）、shell（命令）、review（代码审查）、edit（改动）、ui（前端）、doc（文档）；spawn_subagents 最多 3 路并行
  • 5 任务完全独立并行 — 每个任务拥有独立 agentState、AbortController、MCP 快照，切换会话不中断运行中的任务
  • 内核模块化拆分 — 单文件内核拆分为 15 个独立模块（agent-loop / tool-executor / subagent / policies / edits / stream / tools-registry / prompt / context / run-state / tool-retry / tool-protocol / path / constants / index）
  • 运行时策略引擎 — 读后编辑强制校验、完成门控（in_progress 的 todo 不允许提前结束）、工具自动重试（最多 4 次，250ms 指数退避）
  • 30+ 内置工具 — 文件读写/编辑、Shell、Git 全套、浏览器预览、代码理解全套
  • MCP 扩展重构 — Fetch / Playwright / Windows-MCP 默认启用，Context7 / GitHub 按需启用，支持 per-server 环境变量配置
  • 界面修复 — 主题按钮、右侧边栏实时同步、Agent 输出 UI（Claude/Codex 风格时间线）、后台任务完成通知

Yan Agent · 桌面 Agent

Capabilities

Agent 能做什么

5 任务独立并行 · 30+ 内置工具 · 最多 40 轮迭代直到任务完成

5 任务并行

每个任务完全独立隔离 — agentState、中止控制、MCP 快照互不干扰，切换会话不中断。

精确编辑引擎

old_string 逐字符匹配 + 唯一性校验 + 写后回读验证，改动可靠落地。

子 Agent 系统

spawn_subagent 委派 explore / shell / review / edit / ui / doc 专家，spawn_subagents 最多 3 路并行。

运行时策略

读后编辑强制校验、完成门控、工具自动重试 — 规则不是建议，是硬约束。

代码理解工具

10+ 代码分析工具：符号大纲、引用追踪、依赖分析、项目扫描、索引搜索。

MCP 扩展

Playwright 浏览器自动化、Fetch 网页抓取、Windows 桌面操控，接入任意 MCP 服务。

Workflow

从想法到结果

你说目标，Agent 自己规划、执行、验证 — 5 个任务同时开工

1

描述想法

用自然语言说出你要什么

2

制定计划

自动拆解步骤，Todo 实时可见

3

逐步执行

读文件、改代码、跑命令，工具失败自动重试

4

验证交付

写后回读校验，预览或提交 Git
