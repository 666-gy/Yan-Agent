Yan Agent 1.0.0 正式发布

v1.0.0 2026-07-09

本版本亮点

  • 完整 Agent 内核 — 最多 40 轮工具循环，todo 计划、写后验证、失败重试
  • 17 个内置工具 — 文件读写/编辑、Shell、Git 全套、内置浏览器预览
  • MCP 扩展 — 预装 Playwright、Windows-MCP，支持自定义 MCP 服务
  • Skill 市场 — 51 个精选 Skill（代码 / UI / 网站），一键注入对话
  • 定时自动化 — 后台并行运行，不抢占当前会话
  • 会话持久化 — 工具调用时间线 + apiTrace 跨轮回放
  • 上下文压缩 & 长期记忆 — 长对话不丢工具链，跨会话记住关键事实
  • 现代化 UI — 暗/亮主题、四页侧边栏、系统托盘保活

Yan Agent · 桌面伴侣

动手执行，交付结果

当任务需要 AI 真正去改文件、跑命令、做自动化时，Yan Agent 在本地工作区端到端完成 — 不是给建议，是直接做完。

Capabilities

Agent 能做什么

完整的工具调用循环，最多 40 轮迭代直到任务完成

读写代码

读取、编辑、创建项目文件，写后自动回读验证，改动可靠落地。

Shell & Git

执行命令、查看 diff、提交推送，完整的 Git 工作流内建。

MCP 扩展

Playwright 浏览器自动化、Windows 桌面操控，接入任意 MCP 服务。

内置浏览器

写完网页或小游戏，Agent 自动打开预览 — 所见即所得。

定时自动化

间隔、每日、一次性任务，后台并行运行，不打扰当前对话。

记忆 & 压缩

跨会话长期记忆，长对话自动压缩，工具调用链完整保留。

Workflow

从想法到结果

你说目标，Agent 自己规划、执行、验证

1

描述想法

用自然语言说出你要什么

2

制定计划

自动拆解步骤，Todo 实时可见

3

逐步执行

读文件、改代码、跑命令

4

验证交付

确认结果，预览或提交 Git
