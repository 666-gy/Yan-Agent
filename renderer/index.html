导入自定义 Skill

Skill 是一个 JSON 文件，包含 id、name、desc、prompt 字段，可选 tags（code / ui / web）。导入后可在对话中按 / 调用。

{
  "id": "my-skill",
  "name": "My Skill",
  "desc": "一句话描述",
  "tags": ["code"],
  "prompt": "请帮我完成以下任务：\n\n{{cursor}}"
}

v1.3.0 更新内容

Yan 生态协同

  • 新增 Yan Project Map，展示目录、符号与代码依赖关系
  • 与 Yanxi Code 双向同步工作区，支持冷启动与运行中切换

完整工作台

  • 新增内置终端与内置浏览器，执行和验证无需离开应用
  • 桌面宠物跟随当前任务，展示状态、资源占用并支持停止任务
  • 移动端与桌面端实时同步任务、模型和生成结果

模型与多模态

  • 新增 Kimi K3、Kimi K2.7 Code 等模型与最新价格展示
  • 支持图片输入、生成、编辑、持久化预览和下载
  • 根据模型能力动态开放视觉与图片生成工具

代码理解与任务可靠性

  • 5 个任务独立并行，后台任务保持各自工作区与工具状态
  • 增量代码索引、符号追踪、依赖分析与 AI 项目解读
  • Agent 回复附带可核对的文件变更摘要
