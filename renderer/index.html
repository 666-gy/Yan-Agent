Skill 市场

已安装的自定义 Skill

导入自定义 Skill

Skill 是一个 JSON 文件，包含 id、name、desc、prompt 字段，可选 tags（code / ui / web）。导入后可在对话中按 / 调用。

{
  "id": "my-skill",
  "name": "My Skill",
  "desc": "一句话描述",
  "tags": ["code"],
  "prompt": "请帮我完成以下任务：\n\n{{cursor}}"
}
