v1.3.2 主要更新

更稳的任务内核

  • 修复模型断流、空响应和工具失败后的任务状态错乱，让对话、按钮、桌宠和 Windows 通知保持一致
  • 增加明确的错误收尾、上下文压缩和重复错误熔断，长任务不会悄悄停在“正在工作”

更聪明的工具调用

  • 新增动态能力判断，Agent 会根据目标选择必要的 Skill、MCP、浏览器和电脑操作能力
  • Skill 调用改为输入框内的已安装 Skill 选择器，支持一次任务组合多个技能，减少无关工具调用

工作区与验证体验

  • 任务工作区、终端目录和新建空任务彻底隔离；进入新文件夹后会真正切换 Agent 工作区
  • 内置浏览器成为网页和小游戏的默认验收入口，Computer Use、MCP 与 Playwright 按任务需要才启用

输入框与权限访问

  • 新增推理速度、工作方式和权限访问控制，可在常规、计划、目标及三档审批策略之间切换
  • 完全访问启用前增加确认，并保留文件、网络和高风险系统命令的安全边界
